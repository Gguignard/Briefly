# Epic 8 : Catégorisation Personnalisée

**Epic ID :** epic-08-categories
**Priority :** P1 (High)
**Domain :** User Customization

---

## Objectif

Permettre aux utilisateurs (payants) de créer des catégories personnalisées pour organiser leurs newsletters et filtrer leurs résumés. Les utilisateurs gratuits voient la feature mais ne peuvent pas créer plus de 3 catégories.
